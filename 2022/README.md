# AoC 2022

## Languages

- TypeScript with Deno runtime
- Rust
- Swift

## Commands

### TypeScript

```shell
deno run --allow-read deno/01.ts
```

### Rust

```shell
cargo run --bin 01
```

or

```shell
cargo run --release --bin 01
```

### Swift

```shell
swift swift/01.swift
```

## Inputs

Place all input files at `input/<day>.in`, day in 2 digits format to keep things sorted nicely.
