# Aoc 2021

Rust + Deno :heart:

## Generate a new day with Rust

```bash
cargo generate --git https://github.com/icyJoseph/advent-of-code.git template --name day-1
```

## Run Rust binaries

```bash
cargo run --bin day-1
```

## Run Deno scripts

```bash
deno run --allow-read --allow-write deno/day-1.ts
```
